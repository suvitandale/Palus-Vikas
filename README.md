Palus Vikas - Local Flask app

1. Create a virtual env:
   python -m venv venv
   source venv/bin/activate   # mac/linux
   venv\Scripts\activate      # windows

2. Install requirements:
   pip install -r requirements.txt

3. Initialize DB:
   python init_db.py

4. Run app:
   python app.py

5. Open browser: http://127.0.0.1:5000

Replace the images in static/images/ with real images for header and politicians.
